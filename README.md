# Brain-games

A small set of games for the console. Games are based on solving arithmetic problems.

## Installation

`npm install -g braingamesbyslambe`

## Usage

This package includes following games:

* #brain-balance
* #brain-calc
* #brain-even
* #brain-gcd
* #brain-prime
* #brain-progression

Just enter name of the game in console:

Examples:

`brain-calc`
